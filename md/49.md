# [49. Group Anagrams](https://leetcode.com/problems/group-anagrams/description/)

## Python
```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        res = defaultdict(list)

        for s in strs:
            char_array = [0] * 26
            for c in s:
                char_array[ord(c) - ord("a")] += 1

            res[tuple(char_array)].append(s)

        return res.values()
```
