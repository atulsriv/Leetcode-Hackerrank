# [242. Valid Anagram](https://leetcode.com/problems/valid-anagram/description/)

## Python 

```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        letter_count = {}

        for c in s:
            if c not in letter_count:
                letter_count[c] = 1
            else:
                letter_count[c] += 1

        for c in t:
            if c not in letter_count:
                letter_count[c] = 1
            else:
                letter_count[c] -= 1

        for c in letter_count.keys():
            if letter_count[c] != 0:
                return False

        return True
```
